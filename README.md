# ⌨️ Typing Speed Test (Java Swing)

A modern **Typing Speed Test Application** built using Java Swing.
This project helps users improve typing speed and accuracy with real-time feedback.

---

## 🚀 Features

* ⏱ **60-second timer per word**
* 🎯 **Accuracy calculation**
* ✅ **Live status (Correct / Wrong / Typing)**
* 📊 **Progress bar with color change (Green → Orange → Red)**
* 🔄 **Start / Restart functionality**
* ⌨️ **Auto focus on typing field**
* ↩️ **Press Enter to move to next word**
* 🧠 **Word levels (Easy → Medium → Hard)**
* 🔤 Supports:

  * Lowercase & UPPERCASE words
  * Numbers & alphanumeric
  * Special characters
  * Full sentences

---

## 🖥️ UI Preview

## ▶️ How to Run

1. Clone the repository:

```bash
git clone https://github.com/your-username/typing-speed-test.git
```

2. Open in any IDE (NetBeans / IntelliJ / Eclipse)

3. Compile and run:

```bash
javac TypeSpeed.java
java TypeSpeed
```

---

## 🎮 How to Use

1. Click **Start**
2. Type the displayed word/sentence
3. Press **Enter** to go to next word
4. Timer stops when correct word is typed
5. View final result after completing all words
